# About the project
This project is the round 1 technical test for [Ellty](https://tally.so/r/J9KJQr) 
## Prerequisites
Before setting up the project, you will need the following:
- [`git`](https://git-scm.com/)
- [`make`](https://www.gnu.org/software/make/) | This is **optional**.
- [`npm`](https://www.npmjs.com/) or any Node.js package manager.

## Guide

### Clone the repository
Clone the repository by using this command (alternatively you can use Github Desktop)
```
git clone https://github.com/SwissNuff/task-one.git
```

### Components setup
Navigate to the directory where you have cloned the repository.

#### Frontend

To setup the frontend, run the following commands:
```
cd task-one
npm install
npm run dev
```
